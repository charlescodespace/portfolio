Whitefield FC Football Academy Attendance Tracker

Whitefield FC, Bangalore

Description

Built a full-stack attendance management system for Whitefield FC Football Academy, replacing manual attendance registers with a secure and centralized digital platform.

The application enables coaches to manage players, schedule training sessions, mark attendance, upload session photos, and monitor academy activities through an intuitive dashboard.

Designed using modern web technologies with secure authentication, REST APIs, and PostgreSQL to deliver a scalable, responsive, and efficient solution for academy operations.

Key Features

  • Secure JWT authentication for coaches
  • Player, session, and attendance management
  • Attendance photo upload functionality
  • Dashboard with reports and statistics
  • Responsive UI with modern user experience

Tech Stack

  • Next.js - Frontend framework
  • NestJS - Backend REST APIs
  • PostgreSQL & Prisma - Database management
  • Tailwind CSS - Responsive user interface
  • JWT & Multer - Authentication and file uploads

Impact

  • Eliminates manual attendance tracking
  • Improves academy management efficiency
  • Provides real-time attendance insights and reports
